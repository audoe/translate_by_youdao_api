#有道翻译

使用有道的翻译api实现一个小型的翻译工具，未来会支持百度的API

###使用方法

在有道的 [API申请页] (http://fanyi.youdao.com/openapi?path=data-mode "申请页面")
,申请一个key，相关的内容写入到 **youdao.conf** 这个文件里。

'''python

    python translation.py -k word //work 可以是中文或者英文，有道会判断相应的逻辑


'''

- 有道的网络翻译
